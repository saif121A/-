# بوت حاسبة اللفل والموانئ

البوت يرسل زر في دسكورد. العضو يضغط الزر وتطلع له نافذة يكتب فيها:

- نقاطه الحالية
- اللفل اللي يبي يوصل له
- كم يشيل الميناء الواحد
- كم خبرة تطلع على الحبة

بعدها البوت يرد بقائمة مثل:

```text
بعد ميناء 1:
824,754 + 12,250 = 837,004
تدخل RANK 70 ✅
يبقى لك على 71: 1,260 صندوق
```

## التشغيل

1. فك الضغط.
2. افتح CMD داخل مجلد البوت.
3. اكتب:

```bash
npm install
```

4. انسخ ملف `.env.example` وغيّر اسمه إلى `.env`.
5. عدّل القيم:

```env
DISCORD_TOKEN=توكن_البوت
CLIENT_ID=Application_ID
GUILD_ID=ايدي_السيرفر
EPHEMERAL_REPLY=true
```

- `EPHEMERAL_REPLY=true` يعني النتيجة تظهر للشخص فقط.
- `EPHEMERAL_REPLY=false` يعني النتيجة تظهر للجميع في الروم.

6. سجل الأوامر:

```bash
npm run deploy
```

7. شغل البوت:

```bash
npm start
```

## أوامر البوت

يرسل لوحة الزر:

```text
/rank-panel
```

حساب مباشر بدون زر:

```text
/rank xp:824754 target:75 amount:1225 value:10
```
